Setting up your Raspberry Pi

What you will need

Hardware Cables Extras
Raspberry Pi Power Supply 5V 3.0 amp Case
Micro SD Micro HDMI Headphones
Keyboard HDMI to Micro adapter Ethernet Cable
Mouse HDMI to DVI adapter
HDMI to VGA adapter

Outline of Steps

  1. Set up your SD Card
  2. Connect your Pi
  3. Start up your Pi
  4. Finish the setup
  5. What's next?

Step One: Set up your SD Card

You need your SD card to have Raspbian on it, which is the operating system for the Raspberry Pi. The easiest way to install Raspbian is by downloading a software called NOOBS onto your SD card. To do this you need a computer that has an SD card port.

  • Visit the Raspberry Pi Set up your SD card page follow the instructions.

Step Two: Connect your Raspberry Pi

Now it's time to connect all the hardware to the Pi. It's important to do this in the right order, so that all your components are safe.

  • Visit the Connect your Raspberry Pi page and follow the instructions.

Step Three: Start up your Raspberry Pi

Your Raspberry Pi doesn't have a power switch: as soon as you connect it to a power outlet, it will turn on.

  • Visit the Start up your Raspberry Pi page and follow the instructions.

Step Four: Finish the Setup

When you start your Raspberry Pi for the first time, the Welcome to Raspberry Pi application will pop up and guide you through the initial setup.

  • Visit the Finish the Setup page and follow the instructions.

What's next?

Congradulations! You can now start hacking with the Pi! Check out the Tutorials page of this website for some step by step instuctions.
